# 📧 Spam Email Classifier

This project classifies emails/messages as **Spam** or **Not Spam** using Natural Language Processing (NLP).

## Features

- Text preprocessing
- Stopword removal
- Stemming
- TF-IDF Vectorization
- Multinomial Naive Bayes
- Predict custom emails

## Technologies Used

- Python
- Google Colab
- Pandas
- NLTK
- Scikit-learn

## Dataset

SMS Spam Collection Dataset

## Accuracy

Approximately 97–99% (depends on train-test split)

## How to Run

1. Clone the repository.
2. Install dependencies.
3. Open the notebook.
4. Run all cells.
5. Test with your own email.

## Example

Input:

Congratulations! You have won a free iPhone.

Output:

Spam Email
